# 更新记录

## 1.3

相较于 `1.2`，这一版重点放在「响应式体验整理」和「移动端可用性」上，主要把中间宽度、移动端和案例演示区的布局节奏统一下来，让不同设备下都更顺手、更稳定。

### 新增

- 新增案例演示在中小屏下的横向滑动模式，支持一屏一张切换。
- 新增案例演示分页提示器，方便在移动端和中间宽度下判断当前页。
- 新增移动端 / 中间宽度下颜色选择面板更高的展示比例，取色区域更易操作。

### 调整

- 调整顶部导航在移动端与中间宽度下的布局规则：
  - 深浅模式开关按断点切换位置
  - 工具条改为更稳定的一行或两行结构
  - 基准 / 数量 / 颜色的宽度分配更统一
- 调整 `Playground` 与 `案例演示` 头部在小屏下的排版，让标题说明与切换控件保持单行、两端对齐。
- 调整移动端颜色面板与基准下拉菜单的弹出方向，统一改为向上展开，避免被底部工具条遮挡。
- 调整案例演示在中间宽度下的展示策略，提前进入单卡横滑模式，不再同时露出多张。
- 调整变量预览区视觉风格，参数面板背景改为和页面底层同色，只保留必要层级感。

### 修复

- 修复中间宽度下顶部导航重复出现两个深浅模式开关的问题。
- 修复中间宽度下顶部导航被挤成三行、数量步进区和 HEX 输入区不撑满容器的问题。
- 修复移动端顶部导航滚动时仍被脚本缩窄的问题。
- 修复案例演示卡片左上角裁切不干净、外层壳背景层级过多的问题。
- 修复移动端与中间宽度下多个板块头部频繁换行、切换控件掉到下一行的问题。
- 修复移动端颜色面板、基准下拉菜单在小屏下不易点击或被遮挡的问题。

## 1.2

相较于 `1.1`，这一版重点继续打磨「真实感」和「可用性」，让用户不只是生成色阶，而是更容易判断效果、理解输出、直接拿去用。

### 新增

- 新增更完整的 `APP` 案例演示内容，预演区现在更像真实产品界面，而不是抽象示意。
- 新增预演区更细的组件层级，包括状态标签、统计小卡、文件进度条、设备列表等常见产品元素。
- 新增变量输出格式切换：
  - `清单`
  - `CSS`
  - `JSON`
- 新增变量输出用途说明，区分给设计、前端、工具的不同使用方式。

### 调整

- 调整暗色模式整体配色，按新的 Figma 深色方案更新，不再只是把浅色版压灰。
- 调整案例演示区在浅色 / 深色模式下的对应关系，让深底、亮字、强调色的层级更明确。
- 调整案例演示卡片内容结构，让左、中、右三张卡更接近真实产品中的常见组件组合。
- 调整变量输出区信息架构，不再只展示一大段代码，而是先解释用途，再给不同输出格式。
- 调整复制变量按钮文案和逻辑，改为复制当前所选输出格式。
- 调整案例演示展示策略，当前版本先固定展示 `APP`，隐藏 `表单` 切换入口。

### 修复

- 修复案例演示区深色模式下颜色关系不对、缺少深色感的问题。
- 修复雷达区域 `ME / A / G / R` 四个节点没有跟随全局配色变化的问题。
- 修复案例演示区部分元素虽然换色了，但内部组件关系不真实、参考价值不高的问题。
- 修复变量输出区可读性差、用户难以理解如何使用的问题。

## 1.1

相较于 `1.0`，这一版主要围绕「更像一个可用产品」继续完善，重点补了配色输入效率、场景预览价值和整体交互完整度。

### 新增

- 新增自定义颜色选择器，支持直接在面板里调色，不再只依赖浏览器原生取色。
- 新增 `HEX / HSB / RGB / HSL` 四种颜色数值模式切换。
- 新增吸管入口，方便直接取色。
- 新增 Playground 区域的 `HSL` 显示格式。
- 新增行业场景预览切换：
  - `APP`
  - `Site`
- 新增更完整的场景演示内容，让用户可以直接看当前色阶在真实界面角色里的落位效果。
- 新增全局复制成功/失败提示，让复制色值和复制变量更明确。

### 调整

- 调整色阶算法，让深色 / 浅色模式下的表现更稳定。
- 调整相邻色阶过渡，让颜色连续性更顺，不容易出现跳色和不和谐。
- 调整顶部导航布局、滚动状态、阴影和缩窄逻辑，让页面层级更清楚。
- 调整色阶数量选择逻辑，支持更灵活的数量范围。
- 调整顶部参考色输入方式，让输入、粘贴、替换颜色更顺手。
- 调整取色器交互，默认打开后直接进入 `HEX` 输入，方便用户立即粘贴。
- 调整预览区内容，不再只是说明文案，而是更接近真实产品界面的使用参考。
- 调整 Tooltip、Hover、按钮反馈和细节样式，让整体交互更统一。

### 修复

- 修复多个 Hover 状态反复丢失或触发范围不对的问题。
- 修复数量步进按钮 Hover 区域过大的问题。
- 修复页面滚动有时失效的问题。
- 修复顶部导航滚动时对齐不稳的问题。
- 修复定位图标在 Hover、窗口缩放、滚动时位置不稳定的问题。
- 修复 `HEX` 模式下输入框显示、隐藏和聚焦不正确的问题。
- 修复无效 `HEX` 粘贴后无法快速恢复当前值的问题。
- 修复场景预览和格式切换之间部分显示不一致的问题。
