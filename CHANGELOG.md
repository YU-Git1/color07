# 更新记录

## 1.1

相较于 `1.0`，这一版主要围绕「更像一个可用产品」继续完善，重点补了配色输入效率、场景预览价值和整体交互完整度。

### 新增

- 新增自定义颜色选择器，支持直接在面板里调色，不再只依赖浏览器原生取色。
- 新增 `HEX / HSB / RGB / HSL` 四种颜色数值模式切换。
- 新增吸管入口，方便直接取色。
- 新增 Playground 区域的 `HSL` 显示格式。
- 新增行业场景预览切换：
  - `APP`
  - `Site`
- 新增更完整的场景演示内容，让用户可以直接看当前色阶在真实界面角色里的落位效果。
- 新增全局复制成功/失败提示，让复制色值和复制变量更明确。

### 调整

- 调整色阶算法，让深色 / 浅色模式下的表现更稳定。
- 调整相邻色阶过渡，让颜色连续性更顺，不容易出现跳色和不和谐。
- 调整顶部导航布局、滚动状态、阴影和缩窄逻辑，让页面层级更清楚。
- 调整色阶数量选择逻辑，支持更灵活的数量范围。
- 调整顶部参考色输入方式，让输入、粘贴、替换颜色更顺手。
- 调整取色器交互，默认打开后直接进入 `HEX` 输入，方便用户立即粘贴。
- 调整预览区内容，不再只是说明文案，而是更接近真实产品界面的使用参考。
- 调整 Tooltip、Hover、按钮反馈和细节样式，让整体交互更统一。

### 修复

- 修复多个 Hover 状态反复丢失或触发范围不对的问题。
- 修复数量步进按钮 Hover 区域过大的问题。
- 修复页面滚动有时失效的问题。
- 修复顶部导航滚动时对齐不稳的问题。
- 修复定位图标在 Hover、窗口缩放、滚动时位置不稳定的问题。
- 修复 `HEX` 模式下输入框显示、隐藏和聚焦不正确的问题。
- 修复无效 `HEX` 粘贴后无法快速恢复当前值的问题。
- 修复场景预览和格式切换之间部分显示不一致的问题。
